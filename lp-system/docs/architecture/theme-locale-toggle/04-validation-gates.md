- Navbar exists

- Shell order preserved

- One Navigation component only

- Toggles are semantic interactive elements

- Theme does not create new URLs

- Design token compliance

- Spacing token compliance

- Performance metrics

- Theme persists across reload

- Locale toggle preserves route shape

- No architecture drift

- No new section types

- No navbar ordering changes
