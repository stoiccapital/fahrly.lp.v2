# Typography System

Hierarchy:
- Exactly one H1 (Hero only).
- Each section starts with H2.
- H3 for sub-elements only.

Token usage:
- Headlines use heading tokens only.
- Body uses body token only.
- No custom font weights outside tokens.

Highlighting:
- Max one highlighted phrase per section.
- Highlighting uses color tokens only.

Readability:
- Enforce max line length token.
- Avoid multi-idea sentences in UI copy.
