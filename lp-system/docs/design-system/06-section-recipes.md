# Section Recipes

Canonical order:
Hero → Social Proof → Value Props → Features → Deep Dives → Use Cases → Testimonials → Pricing (optional) → FAQ → Final CTA

Each section includes:
- H2 title
- Optional subtitle
- Content blocks
- Optional CTA group
- Optional media

Density:
- Max 6 items per row desktop
- Max 3 tablet
- Max 1 mobile
- Avoid sections exceeding one viewport height
