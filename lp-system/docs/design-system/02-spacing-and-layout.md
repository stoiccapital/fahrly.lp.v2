# Spacing & Layout System

Grid:
- 12-column grid on desktop.
- Tablet/mobile collapse follows breakpoints.
- No freehand widths or px-based layout.

Section spacing:
- Sections use ONLY section spacing tokens.
- No stacking tokens to exceed max spacing.

Internal spacing:
- Block spacing uses ONLY block tokens.
- Components do not define external margins.

Alignment patterns:
- Left-aligned
- Center-aligned
- Split-grid

Forbidden:
- Mixed alignment types in one section
- Arbitrary container widths
