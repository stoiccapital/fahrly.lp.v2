Purpose:
Enforce deterministic, reusable, Stripe-grade visual consistency across all landing pages.

- Same input -> same output.

Decision hierarchy:
1) Brand consistency
2) Token integrity
3) Grid/layout correctness
4) Visual density
5) Readability
6) Aesthetic preference
